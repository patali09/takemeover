# takemeover
Hosted Zone Takeover Tool

Usuage: 
1. CLone it from repo
2. cd takemeover
3. Give Executable permission to takemeover.sh 
  chmod +x takemeover.sh
4. mv takemeover.sh /usr/bin
5. takemeover.sh [-d domain_name] [-f path_for_domanlist_file]
